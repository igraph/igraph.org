sample_hrg {igraph}R Documentation

Sample from a hierarchical random graph model

Description

sample_hrg samples a graph from a given hierarchical random graph model.

Usage

sample_hrg(hrg)

Arguments

hrg

A hierarchical random graph model.

Value

An igraph graph.
