igraph.drawing.vertex

module documentation

Drawing routines to draw the vertices of graphs.

This module provides implementations of vertex drawers, i.e. drawers that the default graph drawer will use to draw vertices.

Class AbstractVertexDrawer Abstract vertex drawer object from which all concrete vertex drawer implementations are derived.
Class AbstractCairoVertexDrawer Abstract base class for vertex drawers that draw on a Cairo canvas.
Class DefaultVertexDrawer The default vertex drawer implementation of igraph.
