class GradientPalette(Palette):

View In Hierarchy

Base class for gradient palettes

Gradient palettes contain a gradient between two given colors.

Example:

>>> pal = GradientPalette("red", "blue", 5)
>>> pal.get(0)
(1.0, 0.0, 0.0, 1.0)
>>> pal.get(2)
(0.5, 0.0, 0.5, 1.0)
>>> pal.get(4)
(0.0, 0.0, 1.0, 1.0)
