igraph.drawing.colors.AdvancedGradientPalette

class documentation

class AdvancedGradientPalette(Palette):

View In Hierarchy

Advanced gradient that consists of more than two base colors.

Example:

>>> pal = AdvancedGradientPalette(["red", "black", "blue"], n=9)
>>> pal.get(2)
(0.5, 0.0, 0.0, 1.0)
>>> pal.get(7)
(0.0, 0.0, 0.75, 1.0)
